# ExpressJs Backend (MongoDB) 🗄️

A small Express.js backend demonstrating connection to MongoDB. Simple project for learning how to set up a server and connect a database.

## Features
- Basic Express server
- MongoDB connection (example)

## Technologies Used
- Node.js
- Express
- MongoDB

## Project Structure
```
server.js / index.js — entry point
routes/ — API routes (if present)
models/ — database models (if present)
```

## Installation / How to run 🚀
1. Clone the repo.
2. Install dependencies: `npm install`.
3. Set MongoDB URI in env and run: `node server.js` or `npm start`.
